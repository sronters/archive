MedArchive - продукт для автоматической обработки архива прайс-листов клиник-партнеров.

Система принимает PDF, DOCX, XLS, XLSX и сканированные документы, извлекает услуги и цены, связывает строки с единым справочником услуг, отправляет спорные позиции оператору и формирует проверенную базу цен.

## Что уже готово

- API на FastAPI в пространстве `/api/v1`
- загрузка архива и отдельных файлов
- очередь обработки документов
- хранение исходных файлов и происхождения каждой строки
- обработчики XLSX, XLS, DOCX, текстового PDF и граница для OCR PDF
- загрузка и синхронизация справочника услуг
- сопоставление услуг по коду, названию, синонимам и похожести
- очередь ручной верификации
- версионирование опубликованных цен
- экспорт JSON, CSV и XLSX
- вебхуки после публикации
- административная панель на Next.js
- документация API через Swagger/OpenAPI и Mintlify

## Короткая формулировка

MedArchive превращает разнородные прайсы клиник в проверенную базу услуг и цен: система извлекает данные из файлов, нормализует услуги по справочнику, оставляет оператору только спорные позиции и отдает готовые цены через API, экспорт и вебхуки.

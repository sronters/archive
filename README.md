
<h4>
MedArchive is a full-stack system for automated clinic price-list processing. It ingests ZIP, PDF, DOCX, XLS/XLSX and scanned files, extracts services and prices, normalizes them against a unified catalog, sends uncertain matches to operator review, and publishes verified data via REST API, exports and signed webhooks. The project includes FastAPI, Celery, Next.js admin UI, OpenAPI/Swagger contracts and Mintlify (AI-optimized developer documentation)
</h4>
